# Mako Blade

Blade templating engine for Mako Framework.

## Requirements

Mako 9.1 or greater.

## Installation

Configure composer.json:

	"require": {
		...
		"makopro/blade": "*"
	,
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/mako-pro/blade"
        }
    ],
    "minimum-stability": "dev-main"
